# Certification evidence

This directory is for reproducible, externally checkable claims about the Diamond compiler.

Scope:
- Keep evidence bundles, benchmark snapshots, and certification results.
- Keep reproducibility scripts only if they are part of the official certification workflow.

If you are using this as a public evidence folder:
- Reference every claim from `docs/START_HERE.md` and gate docs.
- Ensure outputs are deterministic and timestamped.
- Archive or remove stale artifacts periodically.
